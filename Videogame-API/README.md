# Videogame-API
A REST API for managing videogames, publishers, platforms, categories, and users. Built with **PHP Slim 4** and **Eloquent (Illuminate Database)**, with request validation via **Respect/Validation** and optional authentication using **Bearer tokens** or **JWT**.

## Tech stack
- PHP (Slim 4)
- Composer
- MySQL
- Eloquent ORM (Illuminate Database)
- Respect/Validation
- firebase/php-jwt
